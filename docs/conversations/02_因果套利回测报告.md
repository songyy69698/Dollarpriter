## CTO 级根因分析

### 核心问题：200x 杠杆 + 0.15% 止损 = 每笔亏 ~$15

```
$50 保证金 × 200x = $10,000 仓位
$10,000 × 0.15% 止损 = $15 亏损/笔
$10,000 × 0.04% × 2 = $8 手续费/来回

→ 止损一笔净亏 ≈ $15 + $8 = $23
→ $200 本金经不起连续 8-9 次止损
```

### 模式观察

1. **2月/3月前几天能赚钱**：2月3-5日 +$271.9，3月1-3日 +$241.3
2. **但随后连续大亏日会吃掉全部利润** → 策略缺乏「亏损日提前收手」机制
3. **胜率仅 20-23%**：每 5 笔才赢 1 笔，赢的金额不够覆盖亏的
