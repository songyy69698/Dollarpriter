# 📊 日内交易系统规格书 V1.0

> 2026-03-19 凌晨 CEO × CTO 反覆研讨整理

## 核心原则

**有纪律、有记忆、反覆观察再决定。多空都能赚，不是策略方向的问题，是观察不到位的问题。**

---

## 一、每日观察流程

### 周日/周一早上：周级判断
```
bun run src/sunday-scanner.ts
```
- 过去7天趋势方向（EMA20/50位置）
- RSI周级位置（超卖<30 / 中性 / 超买>70）  
- 波动等级（ATR高/中/低）
- 各窗口过去7天涨跌比
- 输出：本周偏多/偏空/双向

### 每个窗口前：4重共振判断

**4个全绿才进场，任何1个红灯就跳过：**

| # | 观察维度 | 做多绿灯 | 做空绿灯 |
|---|---------|---------|---------|
| 1 | POC方向(前4h) | POC上移>5pt | POC下移>5pt |
| 2 | RSI位置 | RSI < 60 | RSI > 40 |
| 3 | 日振幅已用% | <70%跟势 / >70%反转 | <70%跟势 / >70%反转 |
| 4 | K棒结构 | 近3根无长上影(>40%) | 近3根无长下影(>40%) |

## 三、窗口操作规则

### 08窗口（24pt均振幅）
- 已走<50%: POC方向跟势 + RSI确认
- 已走50-70%: 只有4绿才做

### 15窗口（19pt均振幅，最小）

- 已走>70%: **反转方向**（63%准）

### 22窗口（41pt均振幅，最大）
- **跟POC方向做**（续行62%）
- 振幅大 = 盈利空间最大
- 回调进场 + 窗口收盘出

## 四、进场与出场

### 进场：回调进
- **不在窗口开盘就冲进去**
- 做多: 等前20min内的低点进场
- 做空: 等前20min内的高点进场
- 回测证明：回调进 60%胜率+$3792 vs 开盘冲 39%胜率-$2508

### 出场：窗口收盘平仓
- 不用SL（SL=8在窗口内必被扫）
- 窗口结束时市价平仓
- 最大持仓1小时

## 五、危险信号（不做的条件）

1. **RSI>60要做多** → 不做（超买追顶）
2. **RSI<40要做空** → 不做（超卖追底）
3. **ATR<3** → 不做（波动太小扣手续费亏）
4. **近3根K线连续长上影做多** → 不做（卖压）
5. **连涨>150pt/2天 + RSI>60** → 不追多（趋势疲劳）
6. **周六日 + 振幅<60pt** → 不做

## 八、亏损单根因分析

### 本周亏损单逐笔检讨

#### ❌ 3/15 08窗 做多 -$25
```
环境: ETH从$2075涨到$2094(+19pt) | RSI=64 | ATR=3.8 | 振幅已走56%
进: $2092 → 出: $2088
```
**根因**: RSI=64接近超买，涨势动力已衰。ATR=3.8极低=市场没有波动，任何方向都赚不到钱。
**教训**: RSI>60 + ATR<5 = 双重红灯，绝对不做。✅四重共振已过滤。

#### ❌ 3/15 15窗 做多 -$17
```
环境: RSI=59 | ATR=2.3 | 振幅已走86% | POC+9(微弱)
进: $2108 → 出: $2106
```
**根因**: ATR=2.3极低，窗口只动了10pt，扣手续费$8后必亏。已走86%=振幅空间用完。
**教训**: ATR<3 = 市场死水，不做。✅四重共振已过滤。

#### ❌ 3/16 08窗 做多 -$14
```
环境: ETH前一天暴涨$2075→$2178(+103pt) | RSI=51 | ATR=10.2 | 振幅已走52%
进: $2172 → 出: $2171
```
**根因**: 前一天已暴涨103pt，08窗口是上涨途中的"喘息回调"。POC显示+73(巨大上移)但涨势需要休息。RSI=51刚好没触发过滤（<60），是漏网之鱼。
**教训**: POC涨幅>50pt说明前面已经暴涨，应该警惕不追。需加规则：**POC>50不追多**。

#### ❌ 3/16 15窗 做多 -$58（本周最大亏损）
```
环境: ETH当日从$2094涨到$2268(+174pt)! | RSI已升到55+ | 振幅已走91%
进: $2260 → 出: $2249
```
**根因**: 当日已暴涨174pt（超过日均129pt），振幅91%已用完。在$2268的高点追多=在山顶买入。K棒已出现长上影线（卖压）。
**教训**: 在振幅>日均129pt时不追多。K棒有长上影不追。✅四重共振K棒过滤已挡住。

#### ❌ 3/17 08窗(假设按反转规则做空) -小亏
```
环境: 连涨2天+$199 | RSI=39 | 振幅已走100%
```
**根因**: 趋势太强，连涨2天后仍在涨。反转规则在强趋势中失效。08窗口涨了5pt（几乎没动）。
**教训**: 连涨>150pt/2天时，不用反转规则，改用POC方向。
