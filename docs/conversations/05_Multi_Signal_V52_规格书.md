## V52 六步入场流程

```
Step 1: 找位置 → Volume Profile 标记 POC/VAH/VAL/HVN/LVN
Step 2: 等待价格到关键位 → 没碰到 = 不交易
Step 3: 观察吸收 (Absorption) → 大量打下来但价格不跌 = 做多信号
Step 4: Delta 翻转确认 → 连续 3-5 根 Delta 从负转正 = 买方接管
Step 5: K棒收盘确认 → 收盘在支撑位上方 + 实体足够大
Step 6: 开单 → SL: 关键位后方的 LVN, TP: 下一个 HVN
```

## V52c 三支柱策略（改进版）

### 为什么改？

```
V52a (纯VP+5步): 44% WR, +11.9pt, 但只有 25笔/5月 (太少)
V52b (放宽VP):   32% WR, -88.2pt, 314笔/5月 (亏损)

问题: VP HVN edge 信号太少（严格版）或太不精确（宽松版）
解决: 用 VWAP 做主要动态支撑压力，VP 只做背景参考
```

### 三支柱

```
① VWAP Bounce/Rejection → 开点位 + 频率（每天 3-8 次碰 VWAP）
② CVD Divergence → 判方向（比单纯 delta 趋势更准确）
③ VP 结构 → 背景环境 + SL/TP 目标
```

### VWAP Bounce（做多进场）

```
条件:
  ① 趋势: 价格在 VWAP 上方运行（看多）
  ② 回拉: 价格回到碰 VWAP（或略穿 VWAP 1-2pt）
  ③ 拒绝: K 棒出现下影线碰 VWAP + 收盘在 VWAP 上方
  ④ 确认: 下一根 K 棒开盘 > VWAP → 进场

SL: 拒绝 K 棒的最低价 - 1pt
TP: 依 VP 结构的下一个 HVN，或 R:R 2x
```

### VWAP Rejection（做空进场）

```
条件:
  ① 趋势: 价格在 VWAP 下方运行（看空）
  ② 拉升: 价格回到碰 VWAP（或略穿 VWAP 1-2pt）
  ③ 拒绝: K 棒出现上影线碰 VWAP + 收盘在 VWAP 下方
  ④ 确认: 下一根 K 棒开盘 < VWAP → 进场

SL: 拒绝 K 棒的最高价 + 1pt
TP: 依 VP 结构的下一个 HVN，或 R:R 2x
```

### CVD Divergence

```
看多背离: 价格创新低 + CVD 没跟创新低 → 买方建仓
看空背离: 价格创新高 + CVD 没跟创新高 → 卖方派发
```
