## 出场逻辑重写

**删除：**
- Layer A: 惯性校验（1.5s/0.05%）
- Layer C: 百分比保本锁定（0.12%）
- Layer F: 保本后回落

**新增：**
- **硬止损**：`pnlPt <= -SL_POINTS`（-8 点）→ 立即平仓（不受 MIN_HOLD_MS 限制）
- **固定止盈**：`pnlPt >= TP_POINTS`（+12 点）→ 立即平仓
- **最低持仓保护**：持仓 < 15 秒时，只有硬止损才能触发平仓

**保留：**
- Layer D: 放量倒货止盈（但需满足 MIN_HOLD_MS）
- Layer E: 效率衰竭止盈（但需满足 MIN_HOLD_MS）
